# ☕ Domain-Driven Design Using Java - FIAP

Bem-vindo ao repositório da matéria **Domain-Driven Design Using Java** da FIAP! 🚀

Aqui você encontrará materiais, anotações e projetos relacionados ao **Domain-Driven Design (DDD)** aplicado em **Java**, abordando conceitos fundamentais para modelagem de software baseada no domínio do negócio.

## 📌 Sobre a disciplina

A disciplina **Domain-Driven Design Using Java** foca na construção de sistemas escaláveis e bem estruturados através da modelagem orientada ao domínio. Entre os principais tópicos abordados, estão:

- 🏗️ **Conceitos Fundamentais de DDD**
- 📦 **Entidades, Value Objects e Aggregates**
- 🔄 **Repositórios e Serviços**
- 🏛️ **Camadas da Arquitetura (Application, Domain, Infrastructure)**
- 📝 **Event Storming e Modelagem Colaborativa**
- ☕ **Implementação em Java com Spring Boot**
- ⚡ **CQRS e Event Sourcing**
- 🔗 **Integração com APIs e Microsserviços**

## 📂 O que você encontrará aqui?

- 📑 **Anotações** sobre os conceitos e práticas aprendidas.
- 💻 **Projetos e implementações práticas** em Java.
- 🔍 **Exemplos e estudos de caso** aplicados ao mercado.
